### Telegram-бот для получения текущей погоды любого города мира.

Данные о погоде получаются с https://openweathermap.org/

Бот реализован на основании статьи https://habr.com/ru/companies/selectel/articles/734194/

Для запуска бота необходимо создать файл .env и разместить в нем токены доступа к Telegram API (можно запросить у @BotFather) и сервису https://openweathermap.org/

```sh
BOT_TOKEN=your_token_for_telegram
OPENWEATHER_TOKEN=your_token_for_openweathermap
```
